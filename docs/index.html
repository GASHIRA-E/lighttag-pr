LightTag PR

コードレビューの質と効率を向上させる Chrome拡張

GitHubのPRコメントに「観点ラベル」を簡単追加

チーム内で一貫したレビュー観点を共有し、コードレビューの質を向上

Chrome ウェブストアで入手 GitHub

主な機能

ラベルの追加

レビューコメント入力欄にさまざまなラベルを挿入するUIを提供します。

複数ラベル対応

複数ラベルを同時付与可能。例：[設計][FYI] コメント内容

ツールチップ表示

ラベルにマウスオーバーした際、説明文をツールチップで表示します。

チーム設定共有

観点リストをJSONでカスタマイズ・チーム共有できます。

使い方

1

コメント入力欄の上の「ラベルを追加」ボタンをクリック

2

ドロップダウンからラベルを選択（複数選択可能）

3

「挿入」ボタンでコメント先頭にラベルを追加

4

コメントを送信
